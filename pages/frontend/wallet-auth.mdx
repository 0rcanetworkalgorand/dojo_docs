# Wallet Authentication

The Dojo uses Algorand wallet connection as its sole authentication mechanism. No passwords, no email — just cryptographic identity.

## Provider Setup

The app wraps the root layout with `@txnlab/use-wallet-react`:

```tsx
import { WalletProvider, useInitializeProviders, PROVIDER_ID } from '@txnlab/use-wallet-react';

const providers = useInitializeProviders({
  providers: [
    { id: PROVIDER_ID.PERA },
    { id: PROVIDER_ID.DEFLY },
  ],
  nodeConfig: {
    network: 'testnet',
    nodeServer: 'https://testnet-api.4160.nodely.io',
  },
});
```

## Using the Wallet

```tsx
import { useWallet } from '@txnlab/use-wallet-react';

function MyComponent() {
  const { activeAccount, providers } = useWallet();

  // activeAccount?.address → 58-char Algorand address
  // providers[0].connect() → triggers wallet popup
}
```

## Auth Guard Hook

Protected pages use `useAuthGuard()` to redirect unauthenticated users:

```tsx
const { isAuthenticated, isLoading } = useAuthGuard();
// Redirects to "/" if no wallet connected
```

## Transaction Signing

When creating escrow stakes or other on-chain actions, the frontend uses the wallet's `signTransactions` method:

```tsx
const { signTransactions, sendTransactions } = useWallet();

const signedTxns = await signTransactions([encodedTxn]);
const { id } = await sendTransactions(signedTxns);
```

The wallet popup appears for user confirmation before any ALGO leaves their account.
